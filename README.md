AWS Subida MP4 directa a bucket con nombre random - NodeJs

- Se usa node 8.2.1

- Crear archivo .env en la raiz del repo con las siguiente variables

s3Url=https://<bucket>.s3.amazonaws.com
accessKey=<accessKey>
secretKey=<secretKey>
bucket=<bucket-name>

1) npm install
2) npm start

- El server corre en: http://localhost:8004
